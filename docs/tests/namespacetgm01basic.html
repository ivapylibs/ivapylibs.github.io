Detailed Description

Simple test of single Gaussian foreground modeling using random data.

Defines the mean and sigma value using hard coded quantities and generates random matrix data consistent with the values. The random matrix created consists of not foreground values to which a submatrix is replaced with foreground values.

Generally exists to test out the implementation and debug silly syntactical problems (due to me learning python as I go).
