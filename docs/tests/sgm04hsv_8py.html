Detailed Description

Use Realsense camera to collect background model data, then apply for foreground detection using OpenCV GMM implementation.

Works for D435i or equivalent RGB-D camera from Intel Realsense line.

Needs Intel Realsense D435 or equivalent RGBD camera.

Just run and prints out calibrate process information. Once calibrated, runs another loop to apply to live image stream. Hit "q" to quit.
