Detailed Description

Load stored red glove model and apply.

Display outcomes to window for confirmation that glove tracking is working as expected.

The data is loaded from a bag file given in the command line. This file will initially test out basic parts, then get more complex. Basic versions may be copied over to another file. In principle, they shouldn't need to exist since unit tests elsewhere already provide this type of functionality. Either in Surveillance proper or in Mary_ROS/mary_perception and mary_calibration.

Execution: /// Needs Intel Realsense D435 or equivalent RGBD camera.

./process_bag bagfile

Just run and it displays the color stream to a window.
