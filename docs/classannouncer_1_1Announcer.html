Detailed Description

Base/abstract announcer class.

An announcer packages a signal into text for reporting purposes. The python logging API does not really have an analog to this because a logger is given the text to output. Here, the signal is given and it must be converted to text. The conversion may require specific formatting needs to render it interpretable by a follow-up post-processing routine, or it might even not be text. An overloaded announcer may package it up differently for the channel to output, based on the channel design (it may serialize data provided in a specific format as in the case of ROS messages).

The base announcer relies on function pointers that perform signal translation.
