Detailed Description

Class instance for extracting bright/neon magenta targets, which tend to lie in a corner region of the RGB color cube.

This color is relatively unique in natural and built environments. For natural environments, it tends to be rate. Built environments might be morelikely to have the color, but it is still rare. Especially in small environments with low scene complexity.
